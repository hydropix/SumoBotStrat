Bruno utilise un shield sensor pour le Mega. Les capteurs se branchent sur les rangées S (signal) / V (VCC) / G (GND) du shield.

**Why:** Simplifie le câblage, pas besoin de breadboard ni fils Dupont séparés pour VCC/GND.
**How to apply:** Toujours indiquer la rangée du shield (ex: "rangée A0 du shield") plutôt que des branchements fils individuels.
